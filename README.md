System preparation tool

  - Setup Git https://git-scm.com/downloads
  - Download and install node from https://nodejs.org/en/download/
  - Install 'n' with *npm install -g n*
  - Install node version 7.6.0 *n 7.6.0*

Installation

  - Clone repo
  - cd mystorybook
  - npm install

Run Storybook

  - npm start
  - npm run storybook

Run Tests (Optional)

  - npm run test
